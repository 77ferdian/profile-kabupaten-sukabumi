# Profil Kabupaten Sukabumi

Sebuah proyek halaman web statis (HTML & CSS) yang menampilkan profil, pesona alam, sejarah, dan informasi geografis mengenai Kabupaten Sukabumi, Jawa Barat. Proyek ini dibangun dengan desain yang responsif dan menarik.

## 📌 Fitur
- **Beranda (Hero Section):** Menampilkan keindahan alam Sukabumi dengan latar belakang gambar hero dan badge informasi singkat.
- **Sejarah:** Informasi mengenai asal-usul nama Sukabumi dan sejarah berdirinya.
- **Geografis:** Penjelasan tentang letak, luas wilayah, dan kondisi geografis Sukabumi.
- **Wisata Alam:** Memperkenalkan destinasi wisata unggulan seperti:
  - Geopark Ciletuh (UNESCO Global Geopark)
  - Pantai Ujung Genteng
  - Curug Cikaso
  - Situ Gunung Suspension Bridge
- **Profil Singkat (Aside):** Menampilkan ringkasan informasi umum tentang Kabupaten Sukabumi.

## 🛠️ Teknologi yang Digunakan
- HTML5 (Semantik)
- CSS3 (Flexbox, Grid, Variabel, dan desain responsif)
- Google Fonts (Cormorant Garamond & DM Sans)

## 🚀 Cara Menjalankan Proyek
1. Clone repositori ini:
   ```bash
   git clone https://github.com/77ferdian/Profile-Kabupaten-Sukabumi.git
   ```
2. Buka folder proyek:
   ```bash
   cd Profile-Kabupaten-Sukabumi
   ```
3. Buka file `index.html` di web browser pilihan Anda (Google Chrome, Mozilla Firefox, Microsoft Edge, dll).
